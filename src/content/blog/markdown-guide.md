## 什么是 Markdown？

Markdown 是一种轻量级标记语言，由 John Gruber 在 2004 年创造。它的设计哲学是：

> 易读、易写，尽可能接近纯文本。

## 基础语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **粗体**：`**文字**`
- *斜体*：`*文字*`
- ~~删除线~~：`~~文字~~`
- `行内代码`：反引号包裹

### 列表

无序列表：
- 项目 A
- 项目 B
- 项目 C

有序列表：
1. 第一步
2. 第二步
3. 第三步

### 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](path/to/image.jpg)
```

### 代码块

支持语法高亮：

```python
def hello(name: str) -> str:
    return f"Hello, {name}!"

print(hello("Astro"))
```

### 表格

| 特性 | Markdown | HTML |
|------|----------|------|
| 学习成本 | 极低 | 较高 |
| 可读性 | 优秀 | 一般 |
| 编写速度 | 快 | 慢 |

## 写作建议

1. **一篇一个主题** — 保持文章聚焦
2. **善用标题层级** — 不要跳级（比如 # 直接跳 ###）
3. **代码块标注语言** — 方便语法高亮
4. **图片控制大小** — Markdown 原生不支持，HTML `<img>` 标签可控
5. **保持空行** — 段落间、标题前后保持空行
